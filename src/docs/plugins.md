处理器
=======

处理器是针对文件映射过程中对文件内容进行编辑和修改的处理单元

#### 加载处理器

```javascript
var bone = require('bone');
var less = bone.require('bone-act-less');
```

和`node.js`中加载模块的方式不同，`bonefile.js`里使用`Bone`提供的`require`函数加载处理器模块


#### 使用处理器

```javascript
var bone = require('bone');
var less = bone.require('bone-act-less');


```
